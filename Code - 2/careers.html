Careers at Hot Beans Web


Junior Programmer


As a junior programmer, you will assist lead programmers at Hot Beans Web with the intricate aspects of a project. These include, file and folder sorting, project proofreading for possible errors in syntax and any other tasks set by lead programmers.



Database and systems manager


As a database and systems manager at Hot Beans Web, you have the responsibility to manage all accounts and databases on the Hot Beans Web local network. You will have access to all locations on the network and will need to ensure that it runs up to speed during all business hours, making sure to make regular maintenance checks throughout the day.



Senior client consultant


As a senior client consultant at Hot Beans Web, you will be tasked with organising and scheduling all client meetings. During these meetings, it will your task to ensure all the clients needs are documented, making sure that all official paperwork is filled out in due course.
